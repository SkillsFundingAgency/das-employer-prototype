Before you continue

Check your permissions

You must have the employer’s permission to add apprentices.

Check or update permissions in View and manage employer permissions.

If you do not have permission, ask the employer to create a cohort for you.

Check your ILR file

Make sure the ILR file includes:

  • the employer agreement ID - you’ll find this in View employers and manage permissions
  • a unique email address for each learner

Funding

Reservations

If you’ve reserved funds and you’re only adding one apprentice, you’ll get the option to choose an existing reservation or reserve new funds.

If no reservations are set up or you’re adding multiple apprentices, funds will be automatically be reserved for you.

Transfers

If you are using transfer funding, your employer will need to set up a cohort and send it to you first.

Steps to follow

After submitting your ILR file, allow 15 minutes for records to update in this service. Then follow these steps:

  1. Choose whether to create a new cohort or add to an existing one.
  2. Choose the employer.
  3. Select the learners.
  4. Add recognition of prior learning (RPL) and standard options, if relevant

The Add an apprentice or change their status help page has more information.
