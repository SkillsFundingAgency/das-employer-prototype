Select learner from ILR

Before you start

Make sure you have permission from the employer to add apprentices on their behalf. You can View employers and manage permissions in your account.

If the employer has not given you permission, they can still send an empty cohort for you to complete the apprentice details.

What to include in the ILR file

You must submit the mandatory fields in the ILR file as you usually would for each apprentice.

You should also include:

  • The employer agreement ID (find this on your View employers and manage permissions)
  • The learner’s email address

How it works

Each time you submit an ILR file for a learner, you’ll be able to select the apprentice’s details and add them into a cohort.

It can take up to 15 minutes for the data to come from the ILR submission to the apprenticeship service.

You’ll need to add recognition of prior learning (RPL) and standard options where relevant.

Reserved funding

If reservations are not set up, funding for non-levy apprentices will be automatically reserved when you add them.

We still recommend reserving funding in advance, in case funds become unavailable.
